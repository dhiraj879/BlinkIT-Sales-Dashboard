# 🛒 BlinkIT Grocery Sales Dashboard

## 📊 Project Overview
This project analyzes grocery sales data using Power BI to uncover insights on product performance, outlet efficiency, and sales trends.

---

## 🚀 Dashboard Features
- KPI Cards (Total Sales, Total Products, Average Sales)
- Sales by Item Category
- Sales Distribution by Outlet Size
- Sales Trend Analysis
- Top Performing Products
- Interactive Filters (Outlet Size, Location, Item Type)

---

## 💡 Key Insights
- Snack Foods contribute the highest revenue  
- Medium outlet size generates maximum sales  
- Tier 1 locations dominate overall performance  
- Higher MRP items drive strong revenue  

---

## 🛠️ Tools & Technologies
- Power BI  
- Excel  
- Data Cleaning  
- Data Visualization
